## 程序运行说明
### 1. 环境要求

操作系统：Windows
JDK：17+
构建工具：Maven 3.8+ 或 Gradle 7+
数据库：MySQL 8.0+
Node.js：18+
包管理器：pnpm

### 2. 项目结构

project-root
├── backend/             # Spring Boot 后端  
│   ├── src/             # 源码  
│   ├── pom.xml          # Maven 配置  
│   └── application.yaml  # 配置文件  
├── frontend/            # React + Umi Max 前端  
│   ├── src/             # 源码  
│   ├── package.json     # 依赖配置  
│   └── .umirc.ts        # Umi 配置
